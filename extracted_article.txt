Abstrak
Lembaga Dakwah Kampus (LDK) Syahid UIN Syarif Hidayatullah Jakarta memiliki kuantitas anggota aktif yang masif dengan dinamika portofolio organisasi, kepanitiaan, dan prestasi yang sangat tinggi. Kendati demikian, pengelolaan rekam jejak kompetensi digital anggota saat ini masih menghadapi masalah berupa fragmentasi data, dokumentasi portofolio yang tersebar di lembar kerja spreadsheet manual, serta lemahnya mekanisme validasi yang memicu lambatnya birokrasi pembentukan berkas Curriculum Vitae (CV) resmi organisasi. Penelitian ini bertujuan untuk merancang dan membangun Sistem Informasi Keanggotaan Organisasi berbasis web dengan fokus fungsionalitas otomasi Modul Digital CV guna menyelesaikan inefisiensi administrasi tersebut. Metode pengembangan sistem yang digunakan adalah pendekatan Air Terjun (Waterfall) yang mencakup tahapan analisis kebutuhan, perancangan sistem, implementasi konstruksi, hingga pengujian perangkat lunak. Sistem ini diarsitekturi menggunakan model Three-Tier Architecture dengan bahasa pemrograman PHP lewat Framework Laravel, didukung basis data MySQL, serta pengamanan hak akses menggunakan Role-Based Access Control (RBAC) via Spatie Permission yang mengisolasi fungsionalitas 8 aktor pengurus internal (Helper). Pengujian sistem dilakukan dengan metode Black Box Testing untuk memverifikasi fungsionalitas inti. Hasil penelitian menunjukkan bahwa implementasi platform ini berhasil mengintegrasikan 16 tabel basis data relasional ke dalam satu alur kerja yang kohesif. Fitur otomatisasi rendering dokumen (ATS-Friendly) yang dilengkapi penyematan QR Code unik terbukti secara nyata mengeliminasi redundansi pencatatan (double entry) oleh pengurus, memangkas birokrasi verifikasi portofolio manual, serta memfasilitasi anggota untuk mengunduh berkas Digital CV yang kredibel dan tervalidasi secara instan.
Kata kunci: Sistem Informasi, Keanggotaan Organisasi, Digital CV, Laravel, Role-Based Access Control.

Abstract
Lembaga Dakwah Kampus (LDK) Syahid at UIN Syarif Hidayatullah Jakarta has a massive number of active members with high organizational portfolio dynamics. However, the management of members' competence track records currently faces data fragmentation, manual recording on spreadsheets, and a slow bureaucratic process in generating the organization's official Curriculum Vitae (CV). This study aims to design and develop a web-based integrated membership information system with the main feature of a digital CV automation module to overcome these administrative inefficiencies. The software development method used is Waterfall, which includes the stages of requirements analysis, system design, implementation, and testing. This system is built using the Laravel framework (PHP), MySQL database, and Three-Tier Architecture approach. The testing results using the Black-Box Testing method indicate that all system functionalities run well according to the initial design. In conclusion, this information system is able to automate the recording of member portfolio tracks in real-time, simplify the data validation process by administrators, and accelerate the generation of accurate members' digital CVs.
Keywords: Membership, Curriculum Vitae, Automation, LDK Syahid, Waterfall


pendahuluan
Di era transformasi digital saat ini, urgensi persiapan magang dan karier profesional bagi mahasiswa tingkat akhir berada pada tingkat kompetisi yang sangat ketat. Untuk dapat menembus tahap seleksi perusahaan teknologi raksasa multinasional, pelamar dituntut untuk memenuhi standar administrasi yang sangat tinggi. Lembaga Dakwah Kampus (LDK) Syahid merupakan salah satu organisasi kemahasiswaan terbesar di UIN Syarif Hidayatullah Jakarta yang menjadi wadah bagi ratusan mahasiswa dalam membangun pengalaman kepemimpinan dan portofolio prestasi. Sayangnya, tidak sedikit kader LDK Syahid yang gugur di tahap screening awal rekrutmen bukan karena kurangnya kompetensi, melainkan karena format Curriculum Vitae (CV) mereka yang tidak memenuhi standar tata letak mesin pembaca otomatis di industri.
Kondisi tersebut diperparah dengan pain point yang dialami oleh mahasiswa tingkat akhir. Mereka kerap kali kesulitan (struggle) dan menghabiskan banyak waktu hanya untuk mencari template resume yang dapat menembus sistem Human Resources Department (HRD). Selama ini, data pencapaian dan rekam jejak mereka di LDK Syahid masih terdokumentasi secara parsial dalam spreadsheet manual pengurus. Akibatnya, saat dibutuhkan, anggota harus menyusun portofolio dari nol dengan format visual yang justru berisiko tidak terbaca oleh sistem pelacakan pelamar.
Berdasarkan celah krusial tersebut, penelitian ini menawarkan solusi all-in-one career-prep tool melalui Rancang Bangun Sistem Informasi Keanggotaan Terintegrasi. Platform ini bukan sekadar basis data pendataan keanggotaan konvensional, melainkan dirancang sebagai builder portofolio mandiri. Anggota dapat menyusun rekam jejaknya langsung di website tanpa harus berburu template eksternal, sementara sistem akan secara otomatis mengecek kecocokan format Applicant Tracking System (ATS) dan me-render dokumen Digital CV yang sah, kredibel, dan tervalidasi oleh pengurus organisasi.

Sistem Pelacakan Pelamar (ATS) dan Digitalisasi Portofolio
Applicant Tracking System (ATS) merupakan perangkat lunak yang digunakan oleh divisi HRD untuk memindai, menyortir, dan memberi peringkat pada dokumen pelamar secara otomatis berbasis algoritma pencocokan teks [12]. Digitalisasi portofolio berstandar ATS mengharuskan dokumen memiliki struktur hierarki yang jelas, terbebas dari elemen grafis berat, serta menggunakan jenis fon (font) standar agar ekstraksi data oleh parser mesin berjalan sempurna tanpa anomali output [16].

Antarmuka Pengguna (UI/UX) dan Teknologi Pengembangan
Dalam merancang portfolio builder yang intuitif, pendekatan User Interface (UI) dan User Experience (UX) diimplementasikan sebagai best practice tata letak visual [20]. Prototipe antarmuka dikembangkan menggunakan Figma sebelum diimplementasikan ke dalam Framework Laravel (PHP) berbasis arsitektur Model-View-Controller (MVC). Keamanan sistem diatur melalui Role-Based Access Control (RBAC) via Spatie Permission [14], [18].

kesimpulan
Kesimpulan
Penelitian rancang bangun ini berhasil menyusun cetak biru (blueprint) dan purwarupa aplikasi Sistem Informasi Keanggotaan Terintegrasi berbasis web pada LDK Syahid UIN Jakarta. Melalui model Three-Tier Architecture dan pengamanan Role-Based Access Control (RBAC) via Laravel Spatie Permission, sistem mampu mengisolasi hak akses 8 aktor internal secara andal. Implementasi Modul Digital CV terbukti berhasil memotong inefisiensi administrasi, menghilangkan redundansi data (double entry) akibat pengarsipan manual, serta memfasilitasi kader untuk menghasilkan luaran berkas Digital CV cetak otomatis (ATS-Friendly) tervalidasi yang kredibel berkat integrasi fitur verifikasi keamanan QR Code secara instan.

Saran
Demi menjaga keberlanjutan operasional serta perluasan fungsionalitas platform pada masa mendatang, disarankan bagi pengembangan penelitian selanjutnya untuk mengintegrasikan Sistem Informasi Keanggotaan LDK Syahid ini secara langsung dengan sistem pusat data akademik kampus (seperti API AIS UIN Jakarta). Integrasi proaktif tersebut akan mempermudah validasi otomatis sinkronisasi status kemahasiswaan aktif, IPK, dan data kelulusan kader secara real-time tanpa perlu melalui proses input profil manual kembali oleh anggota. Selain itu, konversi platform menuju bentuk Progressive Web App (PWA) dapat dipertimbangkan guna meningkatkan aksesibilitas aplikasi di perangkat seluler [3].
